Brief Introduction :

> This Project was the first project that we learnt to connect branch pages and use png to link webpages.

> The website introduces and talks about the collection of album cards of TWICE, a kpop girlgroup.

Goals for this project :

> The main goals for this project was to be able to create branch pages in the same file folder and be able to link them so the user doesn't get stuck on a page.

> For this project we were also introduced to "Table", "table rows", and "table data".

Likes and Dislikes about this project :

> This project was a really fun one to make as it was also our first partner project.

> The most difficult part about this project was probably to code on just one laptop when you have two people.

> One of the challenging things of this project was also to use the table element, as I sometimes messes up between table row and table data.
