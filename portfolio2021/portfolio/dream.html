Brief Introduction :

> These Projects were some of our first projects, we were just trying to be able to present text on a website, use styling, and place images.

> The recurring dream website has a short story on it, talking about a recurring dream.

> The gopher website has a some brief information on it about gophers and some random facts, along with images and a gif.

> For the recurring dreams project we also went back and edited the text's style when we first approached css styling.

Goals for these projects :

> For the recurring dreams project was mainly on just simply being able to present text and uploading the project onto Github.

> However, we did got a chance to just play around with different font styles you can do using css later on.

> For the gopher project we learnt about styling the websites and how to put images on it.

Likes and Dislikes about these projects :

> Writing the recurring dreams story was really fun.

> However, the first project did lack a bit of content and now looking at it, things could've been styled more and not just words on the website.

> The making of the gopher project was really interesting as I didn't know a lot of this animal before.

> Aligning the piuctures when making this project was one of the biggest challenges but I really liked how it turned out organized at last.
