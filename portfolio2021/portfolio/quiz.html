Brief Introduction :

> This Project was the first project that we used a form and approached the JavaScript language.

> The website is inspired by buzzfeed quizzes and tells you a cuisine you should try based on your boba tea order.

Goals for this project :

> The main goals for this project was to be able to write a form and it's matching JavaScript and to have it actually work.

> For this project we went really creative and completely random with the questions and the outcomes.

Likes and Dislikes about this project :

> This project was a really funny and random one to finish although it was really hard and I met quite a few obstacles.

> One thing that I encountered when writing the code for this website was to MAKE THE JAVASCRIPT WORK. The outcomes kept on glitching and the Javascript didn't work so I was really frustrated.

> I really had a lot of fun making this project as to match the outcomes with the choices people make as they seemed to make sense but at the same time was ridiculously funny.
